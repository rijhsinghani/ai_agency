  <action>
Create the guide-generator Node.js CLI at research/tools/guide-generator/ following the same CommonJS + Commander.js pattern as research/tools/repurposing-script/.

**package.json:**

- name: "guide-generator"
- dependencies: puppeteer (^24.x), marked (^15.x), commander (^14.0.3)
- bin: { "sameer-guides": "./index.js" }
- No @anthropic-ai/sdk needed — guide content is written manually as markdown, not AI-generated at build time

**index.js (CLI entry):**

- --guide slug: Build a single guide by slug (e.g., missed-call-textback)
- --all: Build all 5 guides
- --output dir: Output directory (default: ../../website/guides/ relative to tool)
- --preview: Also generate a 816x400 PNG screenshot of page 1 for landing page preview images
- Flow: read markdown from guides/{slug}.md, call buildHTML(), call renderPDF(), optionally call renderPreview()
- Valid slugs: missed-call-textback, review-automation, monday-pipeline, quote-writer, no-show-killer

**lib/template.js — buildHTML(markdownContent, guideMeta):**

- Use marked.parse() to convert markdown body to HTML
- Wrap in full HTML document with inline CSS matching brand identity exactly:
  - Background: #1A1A1A, text: #EDE9E3
  - Headings: font-family Roc Grotesk (local), Outfit (Google Fonts fallback), system-ui
  - Body: font-family Gilroy (local), DM Sans (Google Fonts fallback), system-ui
  - Accent line: 2px, linear-gradient(90deg, #7B2FBE, #4DD9E8)
  - Tables: background #1E1E1E, border rgba(237,233,227,0.15), header color #4DD9E8
  - Footer CTA: font-size 13px, color rgba(237,233,227,0.6), border-top separator
- CRITICAL CSS for dark PDF: html,body,\* { -webkit-print-color-adjust: exact; print-color-adjust: exact; }
- @page { size: Letter; margin: 0; }
- .page-break { page-break-after: always; break-after: page; } — triggered by --- (hr) in markdown
- table { break-inside: avoid; page-break-inside: avoid; }
- Page dimensions: 816px width (Letter at 96dpi), 1056px min-height, 64px padding
- Logo: small brand icon in top-right corner position (absolute, top 24px, right 32px)
- Configure marked to replace --- (thematic break / hr) with a div.page-break element so each section gets its own page
- guideMeta object: { title, slug, industry } — used for meta tags in HTML head

**lib/renderer.js — renderPDF(htmlContent, outputPath) and renderPreview(htmlContent, outputPath):**

- renderPDF: puppeteer.launch({ headless: true }), page.setContent(html, { waitUntil: 'networkidle0' }), page.pdf({ path, format: 'Letter', printBackground: true, margin: all 0, preferCSSPageSize: true }), browser.close()
- renderPreview: same browser, page.screenshot({ clip: { x: 0, y: 0, width: 816, height: 400 }, type: 'png' }) — captures top portion of page 1 for landing page preview images
- Export both functions

**lib/generator.js — guide metadata registry:**

- Export GUIDES array with 5 objects: { slug, title, industry, opsPackage, roiAnchor }
  - missed-call-textback: "The 30-second save", Salon, missed-call-text-back.md, "5 calls/week x $80 = $400/week"
  - review-automation: "The review machine", Dental practice, review-reputation.md, "30 to 90 reviews = 25-30% more clicks"
  - monday-pipeline: "The Monday morning pipeline", Cleaning service, lead-follow-up.md, "No more Monday scramble"
  - quote-writer: "The quote that writes itself", Law firm, quote-follow-up.md, "Sunday night = 2 hrs; automation = 10 min"
  - no-show-killer: "The no-show killer", Gym, appointment-reminders.md, "3 no-shows/week x $80 = $12,480/year"
- Export getGuide(slug) lookup function

Run `cd research/tools/guide-generator && npm install` after creating files.

**Format for EVERY guide (3 sections separated by --- page break markers):**

Section 1 — Page 1 (The Hook): Max 150 words. One paragraph naming the pain with specific dollar math. Use the industry assigned to this guide. Lead with the cost of NOT having the automation. This is the part people screenshot and share. Include a Reddit community quote only where it strengthens the hook (Claude's discretion per CONTEXT.md).

Section 2 — Page 2 (How It Works): Max 150 words. Plain English explanation. Include a markdown before/after table with specific dollar amounts and time metrics (not percentages or ranges). Columns: blank, Before, After. Rows: time to response/completion, leads/reviews recovered, monthly cost. Per user decision: "5 missed calls/week x $80 = $400/week lost" style math.

Section 3 — Page 3 (DIY Map): Max 200 words. Per locked decision: "Show the map, not the road." Name the REAL tools (n8n, Twilio, Claude API — per user decision, reference actual stack). List 5-7 steps at headline level only. Call out 2-3 specific gotchas that name the complexity without saying "hire me." No code snippets, no screenshots.

**Footer on every guide (after Section 3):**
_Built by Sameer Automations — if you'd rather skip the setup: calendar.app.google/psycao3CrXjGnmk48_

**Guide-specific details:**

1. **missed-call-textback.md** — "The 30-second save"
   - Industry: Salon (hair/nails). NOT HVAC/plumbing (per Phase 9 UAT fix: industry diversity).
   - ROI: 5 missed calls/week x $80 avg appointment = $400/week = $20,800/year lost
   - Source: ops/packages/missed-call-text-back.md
   - Before/after: 2-6 hours response vs. 30 seconds; most leads gone cold vs. ~50% reply; $0 (paid in lost jobs) vs. $200/month system cost

2. **review-automation.md** — "The review machine"
   - Industry: Dental practice
   - ROI: 8 reviews at 3.9 stars vs. 50+ reviews at 4.7 stars; 25-30% more new patient clicks from Google Maps
   - Source: ops/packages/review-reputation.md
   - Before/after: Manual ask (5% response) vs. automated (40% response); 2 reviews/month vs. 8-12/month

3. **monday-pipeline.md** — "The Monday morning pipeline"
   - Industry: Cleaning service
   - ROI: Monday morning = 45 min scrolling texts/emails/voicemails; automated = 2-min summary in inbox at 7 AM
   - Source: ops/packages/lead-follow-up.md (adapt — package is real-time, guide is weekly summary)
   - Before/after: 45 min Monday scramble vs. 2-min review; 3-5 leads fall through cracks/month vs. zero missed; $0 (paid in lost clients) vs. $150/month

4. **quote-writer.md** — "The quote that writes itself"
   - Industry: Law firm (small practice, not Big Law)
   - ROI: Sunday night = 2 hours writing proposals; automated = 10-minute voice memo to formatted quote
   - Source: ops/packages/quote-follow-up.md
   - Before/after: 2 hrs/quote vs. 10 min; 1-2 quotes/day vs. 4-5/day; lost evenings vs. sent before dinner

5. **no-show-killer.md** — "The no-show killer"

   - ROI: 3 no-shows/week x $80/session = $240/week = $12,480/year
   - Source: ops/packages/appointment-reminders.md
   - Before/after: No reminder (20% no-show) vs. 48hr + 2hr sequence (5% no-show); 3 empty slots/week vs. 0-1; lost revenue vs. $100/month system cost
