**ops/sop/recording-checklist.md** — Three-section checklist with checkbox items (use `- [ ]`). Structure:

**Pre-shoot (day before or morning of)**:

- [ ] Outline is complete — every section filled in, opening line rehearsed
- [ ] Screen recording software is open and tested (OBS, QuickTime, or equivalent)
- [ ] Portfolio piece / demo system is logged in and working — test the workflow trigger before filming
- [ ] Background is clean and consistent with previous videos (same location, same setup)
- [ ] Lighting is adequate — face visible, no harsh shadows, no window backlight
- [ ] Audio is tested — record 10 seconds, play back, check for hum, echo, or background noise
- [ ] Microphone positioned correctly — 6-10 inches from mouth, off-axis from any fans/AC units
- [ ] Water on desk (keep it off camera — move before filming)
- [ ] Phone on Do Not Disturb
- [ ] Camera / screen resolution set: 1080p minimum (4K preferred for post-production crop flexibility)
- [ ] Backup recording option confirmed (phone or second device) if primary fails

**During shoot**:

- [ ] State the video title/topic before starting — helps in editing to locate the "real" take
- [ ] Film the hook 3 times minimum — pick the best take in editing
- [ ] Pause 2-3 seconds between sections — gives clean edit points
- [ ] Speak to one specific person (the business owner in the problem statement), not a generic audience
- [ ] Say the mid-video CTA on camera: "If you want help with this, I have a link below to book a free 15-minute call"
- [ ] Film the demo/screen recording separately from the talking-head segments
- [ ] Record transitions: "Let me show you what that looks like" before switching to screen
- [ ] End screen: look directly at camera, slow down, clear CTA delivery
- [ ] Record a "backup hook" — a second version of the opening in case the first doesn't feel right

**Wrap**:

- [ ] Review footage — confirm hook, demo, results, and CTA sections are all captured
- [ ] Export raw files to organized folder: `content/videos/raw/[video-slug]/`
- [ ] Label screen recordings separately from talking-head files
- [ ] Back up raw footage to external drive or cloud before starting edits

**ops/sop/editing-checklist.md** — Tool-agnostic editing checklist. Items describe outcomes, not software menus. Structure:

**Assembly cut**:

- [ ] Import all raw files (talking-head + screen recordings) into a single project
- [ ] Sync audio if recorded to a separate device
- [ ] Label clips by section: hook-take-1, hook-take-2, demo-screen, results, cta
- [ ] Rough assembly in section order — do not cut anything yet, just sequence
- [ ] Watch the full rough cut once through, no edits — identify what is missing

**Cut review**:

- [ ] Remove all dead air, filler words, and repeated sections (keep pacing tight)
- [ ] Remove the "state the topic" clip from the beginning (this was for your reference only)
- [ ] Trim pauses between sections to 0.5-1 second
- [ ] Color grade or apply consistent LUT to all talking-head clips
- [ ] Screen recordings: match audio levels to talking-head segments

**Captions and subtitles**:

- [ ] Generate captions (auto-captions via editing software or upload to YouTube for auto-gen)
- [ ] Review caption accuracy — correct any misheard words, especially "Sameer," "automation," and any client names
- [ ] Captions match the audio — no lines out of sync

**Thumbnail**:

- [ ] Thumbnail follows brand guidelines: dark background (#1A1A1A), Roc Grotesk Bold text, no more than 5 words
- [ ] If before/after split: left = problem (crossed-out or stressed visual), right = result (clean, numbers)
- [ ] Thumbnail does not duplicate the title — it complements it
- [ ] Export thumbnail as JPG at 1280x720 minimum

**Audio review**:

- [ ] Normalize audio levels — consistent across all clips
- [ ] Check for background noise spikes during scene transitions
- [ ] Final listen with headphones — if anything would make you skip, cut it

**Final check before export**:

- [ ] Two CTAs placed: mid-video at ~60% mark and end screen in final 15 seconds
- [ ] End screen has clickable CTA element (card or overlay) with booking link
- [ ] Video duration matches target (8-10 min for Video 1) — within 90 seconds of target
- [ ] Watch the first 30 seconds one final time — this determines 70% of your audience retention

**Export**:

- [ ] Export at 1080p minimum (H.264 MP4 preferred for YouTube upload speed)
- [ ] File named: `[video-slug]-final.mp4`
- [ ] File saved to `content/videos/edited/[video-slug]/`
