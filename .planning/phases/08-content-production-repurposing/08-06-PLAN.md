<!-- Excalidraw JSON format -->

Excalidraw files are JSON with this structure:

- type: "excalidraw"
- version: 2
- elements: array of shapes (rectangle, arrow, text, ellipse, diamond, line)
- appState: contains viewBackgroundColor, theme, etc.
- Each element has: id, type, x, y, width, height, strokeColor, backgroundColor, fillStyle, strokeWidth, text (for text elements), fontSize, fontFamily, textAlign, points (for arrows/lines)
- Arrow bindings use startBinding/endBinding with elementId to connect to shapes
- fontFamily: 1 = Virgil (hand-drawn), 2 = Helvetica, 3 = Cascadia

<!-- Brand colors for Excalidraw -->

- Background: #1A1A1A (dark base)
- Text: #F5F5F5 (soft white)
- Primary accent: #7B2FBE (purple)
- Secondary accent: #4DD9E8 (cyan)
- Success/result: #10B981 (green)
- Warning/problem: #EF4444 (red)
