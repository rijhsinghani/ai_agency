Create `content-engine/n8n-workflows/platform-publishers/instagram.json`.

**Workflow design:**

Trigger: Supabase Trigger on `content_bank` table for status=approved

Filter: Check that `visual_assets.clips` is not empty (need a 9:16 clip for Reels)

**Instagram Reels two-step upload flow:**

1. Get the best clip from `visual_assets.clips` (highest strength_score)

2. Generate GCS signed URL for the clip with 3600-second expiry
   - n8n Code node calling GCS signed URL API
   - CRITICAL: expiry must be at least 3600 seconds — Instagram takes 30-120s to process after container creation; a short-lived URL will 403 during Instagram's download

3. Step 1: Create media container
   - n8n HTTP Request: `POST https://graph.facebook.com/v22.0/{ig_user_id}/media`
   - Params: `{media_type: "REELS", video_url: signed_url, caption: instagram_caption, access_token: meta_access_token}`
   - Store response `id` as `container_id`

4. Step 2: Poll for FINISHED status
   - n8n Wait node (30 seconds initial wait)
   - n8n HTTP Request: `GET https://graph.facebook.com/v22.0/{container_id}?fields=status_code&access_token={token}`
   - n8n IF node: if status_code == "FINISHED" → proceed; else wait and retry (up to 10 retries, 15s between)
   - If still not FINISHED after 10 retries: log error and skip publishing (don't fail workflow)

5. Step 3: Publish container
   - n8n HTTP Request: `POST https://graph.facebook.com/v22.0/{ig_user_id}/media_publish`
   - Body: `{creation_id: container_id, access_token: meta_access_token}`
   - Store response `id` as `instagram_post_id`

6. Update content_bank: store instagram_post_id in `visual_assets.instagram_post_id`

**Caption format:**
Use `platform_angles.instagram` from content_bank. Append booking CTA:

```
{instagram_caption}

Book a free 15-min discovery call: https://calendar.app.google/psycao3CrXjGnmk48
```
