## Out of Scope

| Feature                          | Reason                                                                     |
| -------------------------------- | -------------------------------------------------------------------------- |
| Full application development     | Too complex, not the play — discrete workflow automations only             |
| Content generation services      | Focus is business process automation, not writing/content                  |
| SEO services                     | Saturated market, not automation, doesn't play to strengths                |
| Website builds for clients       | Wrong positioning — agency is automation layer, not web dev                |
| Enterprise clients               | Long sales cycles, procurement overhead, solo can't compete                |
| No-code DIY training for clients | Trains clients out of needing you, destroys retainer model                 |
| White-label reselling            | Quality control issues, dependency — build everything yourself             |
| Performance-based pricing        | Ties revenue to outcomes you don't control — use value-based fixed pricing |
| Hiring/team building             | Solo operator for v1, expand only after product-market fit                 |
| Paid advertising                 | Only after organic traction validates messaging — no cold spend            |
| Full legal setup (LLC, etc.)     | Form entity when revenue exists to protect — not a Day 1 requirement       |
| Pre-building automation stack    | Market-first: don't spend on tools until client pays                       |
