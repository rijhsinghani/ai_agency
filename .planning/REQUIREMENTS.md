# Requirements: AI Automation Agency

**Defined:** 2026-02-27
**Core Value:** Local service businesses save real money and recover real revenue through AI-powered automations they couldn't build themselves

## v1 Requirements

Requirements for initial launch. Each maps to roadmap phases.

### Brand & Foundation

- [ ] **BRAND-01**: Agency has a name, registered domain, and branded email address
- [ ] **BRAND-02**: Agency has minimal visual identity (logo, color palette, typography)

### Landing Page

- [ ] **LAND-01**: Single-page landing page communicates value proposition in non-technical language
- [ ] **LAND-02**: Landing page displays productized automation packages with published pricing
- [ ] **LAND-03**: Landing page shows ROI value examples (e.g., "recover 8 lost leads/week")
- [ ] **LAND-04**: Landing page has one primary CTA: book a discovery call
- [ ] **LAND-05**: Landing page includes the PoC case study as social proof
- [ ] **LAND-06**: Landing page built using Gemini 3.1 Pro MCP for design assistance

### Core Product (PoC)

- [ ] **POC-01**: Lead follow-up automation is built and working (new lead → AI qualifies → books appointment → sends confirmation)
- [ ] **POC-02**: Automation uses N8N + Claude/LLM + Twilio/Resend + Cal.com stack
- [ ] **POC-03**: PoC is documented as a case study with before/after metrics (response time, leads captured, estimated revenue recovered)
- [ ] **POC-04**: PoC has a video walkthrough recorded (screen recording + narration, 3-5 minutes)

### Productized Packages

- [ ] **PKG-01**: Pre-built N8N automation templates exist for 3-5 high-value use cases common to local service businesses
- [ ] **PKG-02**: Each template has a fixed published price (aligned with $1,500-3,000 range)
- [ ] **PKG-03**: Custom build option is available for engagements beyond template scope
- [ ] **PKG-04**: Template catalog includes: Lead Follow-Up, Missed Call Text-Back, Review Request, Appointment Reminders (minimum 3)

### Lead Generation

- [ ] **LEAD-01**: Cold outreach system exists: email/DM templates personalized to local businesses
- [ ] **LEAD-02**: Local business prospecting list built (Google Maps / Yelp scraping for target area)
- [ ] **LEAD-03**: Free audit offer created: "I found 3 things costing you leads" personalized to prospect's business
- [ ] **LEAD-04**: Outreach cadence defined (Day 1 intro, Day 3 follow-up, Day 7 value-add)

### Content & Marketing

- [ ] **CONT-01**: YouTube channel pivoted from photography to AI automation for business owners
- [ ] **CONT-02**: Channel has updated branding, about section, and banner reflecting new direction
- [ ] **CONT-03**: First 5 video topics planned (mix of: PoC demo, explainer, case study, short-form)
- [ ] **CONT-04**: First 5 videos produced and published
- [ ] **CONT-05**: All content speaks to business owners (non-technical), not builders/developers

## v2 Requirements

Deferred to after first client or first revenue.

### Sales Infrastructure

- **SALE-01**: Discovery call booking (Cal.com) with value discovery script
- **SALE-02**: Proposal template with 3-tier pricing (Starter/Professional/Growth)
- **SALE-03**: Client agreement template (scope, deliverables, 50/50 payment, IP, liability, retainer, exit clause)
- **SALE-04**: Invoicing system via Stripe (one-time build fees + recurring retainer billing)

### Client Delivery

- **DELV-01**: Client onboarding playbook (first 30 days: kickoff, milestones, go-live review)
- **DELV-02**: Monthly retainer reporting template (automations active, triggers fired, value delivered)
- **DELV-03**: Client-readable automation status dashboard or log

### Advanced Automations

- **AUTO-01**: AI website chatbot offering for clients (trained on business FAQ, hours, services)
- **AUTO-02**: Quote/estimate follow-up automation (3-touch sequence after sending estimate)
- **AUTO-03**: Post-job upsell/nurture sequences (seasonal follow-up recommendations)
- **AUTO-04**: Voice AI for inbound calls (AI answers, qualifies, books or routes)

### Growth

- **GROW-01**: LinkedIn presence optimized for local business owner audience
- **GROW-02**: Referral partner pipeline (web designers, bookkeepers, accountants)
- **GROW-03**: Weekly business health report (auto-generated for retainer clients)

## Out of Scope

| Feature                          | Reason                                                                     |
| -------------------------------- | -------------------------------------------------------------------------- |
| Full application development     | Too complex, not the play — discrete workflow automations only             |
| Content generation services      | Focus is business process automation, not writing/content                  |
| SEO services                     | Saturated market, not automation, doesn't play to strengths                |
| Website builds for clients       | Wrong positioning — agency is automation layer, not web dev                |
| Enterprise clients               | Long sales cycles, procurement overhead, solo can't compete                |
| No-code DIY training for clients | Trains clients out of needing you, destroys retainer model                 |
| White-label reselling            | Quality control issues, dependency — build everything yourself             |
| Performance-based pricing        | Ties revenue to outcomes you don't control — use value-based fixed pricing |
| Hiring/team building             | Solo operator for v1, expand only after product-market fit                 |
| Paid advertising                 | Only after organic traction validates messaging — no cold spend            |
| Full legal setup (LLC, etc.)     | Form entity when revenue exists to protect — not a Day 1 requirement       |
