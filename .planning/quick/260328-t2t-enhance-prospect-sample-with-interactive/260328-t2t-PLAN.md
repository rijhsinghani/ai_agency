13. **Scroll animations**: Use a single inline `<script>` with IntersectionObserver. Elements with class `fade-up` start at opacity:0 translateY(20px) and animate to visible on scroll. Progress bars animate width from 0 to target. Donut charts animate from 0 to target angle.
