## Usage Notes

- YouTube allows up to 500 characters for channel tags. Prioritize the first 15-20 tags.
- Paste tags as comma-separated values in YouTube Studio > Customization > Basic Info > Keywords.
- Avoid repeating root words in consecutive tags (YouTube de-weights near-duplicates).
- Revisit every 90 days using YouTube Analytics Search Terms report to identify new organic search terms.
